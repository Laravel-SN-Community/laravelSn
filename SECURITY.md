# Politique de sécurité

## Versions supportées

Seule la dernière version majeure du projet reçoit des correctifs de sécurité. Les versions antérieures ne sont pas maintenues.

| Version | Supportée |
| ------- | --------- |
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Signaler une vulnérabilité

La sécurité de Laravel Sénégal et de ses utilisateurs est importante pour nous. Si tu découvres une vulnérabilité, **merci de ne pas l'ouvrir publiquement** sur GitHub Issues, Discord, WhatsApp ou tout autre canal public.

À la place, envoie un email à : **`security@laravel-sn.com`**

Inclus si possible :

- Une description claire de la vulnérabilité
- Les étapes pour la reproduire
- L'impact potentiel
- Toute proposition de correctif

## Engagement de réponse

Nous nous engageons à :

- **Accuser réception** de ton signalement sous 48 heures ouvrées
- **Évaluer et confirmer** la vulnérabilité sous 7 jours
- **Communiquer un correctif** ou un plan d'action sous 30 jours pour les vulnérabilités critiques
- **Te créditer** publiquement (si tu le souhaites) une fois le correctif déployé

## Programme de divulgation

Nous suivons un modèle de **divulgation coordonnée** :

1. Tu nous signales la vulnérabilité en privé
2. Nous travaillons ensemble pour la corriger
3. Une fois le correctif déployé en production, nous publions un avis de sécurité (Security Advisory) sur GitHub
4. Tu peux alors publier les détails techniques si tu le souhaites

## Hall of Fame

Nous reconnaissons publiquement les chercheurs en sécurité qui nous ont aidés. La liste est maintenue dans [SECURITY-CREDITS.md](SECURITY-CREDITS.md).

## Hors-périmètre

Les éléments suivants ne sont **pas considérés comme des vulnérabilités** dans le cadre de ce programme :

- Attaques nécessitant un accès physique à la machine de la victime
- Attaques d'ingénierie sociale contre nos contributeurs
- Vulnérabilités dans les dépendances tierces (signale-les directement aux mainteneurs concernés)
- Problèmes liés à des configurations utilisateur incorrectes
- Spam, phishing visant nos utilisateurs hors de la plateforme
